# Language Standards

- ALL code comments MUST be in English — no exceptions
- ALL documentation files (markdown, docs/, READMEs, postmortems) MUST be in Spanish
- User-facing text (UI labels, error messages shown to users) MUST be in Spanish
- Variable names, method names, class names, and interfaces in English
- Domain terms that are Mexican financial concepts keep original names: clabe, pagos_stp, facturas, RFC, CURP, SAT
